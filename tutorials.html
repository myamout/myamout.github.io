Setting Up A Bash Terminal On Windows 10

Intro

Having a bash terminal is why many software developers choose OS X or some flavor of Linux, like Ubuntu, to use as their main software development machine. A bash terminal allows for the use of package managers, easily navigating system, files, and is used greatly in full-stack development when testing servers before deploying the code into production. Windows has always lacked a bash terminal simply because Windows is not Unix based. However, now on the latest update of Windows 10 Microsoft was added the ability to enable a bash terminal for users to use. Learning how to use a bash terminal is a very useful skill in development and will give you a major leg up in the job market. Since most tutorials on this site will take advantage of the bash terminal I would like to go ahead and help anyone following along on a Windows machine setup the Windows Linux Subsystem. If you are using a Mac you already have a bash terminal on your machine and if you are using a flavor of Linux then you probably already know how to use the terminal!

Updating To Windows 10

Firstly we need to update to the latest version of Windows 10, the Anniversary Update. To check if you need to update in the Cortana search bar along the bottom left of the taskbar search "updates" and then click on the first app to show up. Now at the top of the page if it says something along the lines of "your system is up to date" you are good to go! If there is a system update waiting for you, the Anniversary Update's code is 1607, then update and resart your computer.



Enabling Windows Subsystem For Linux

Now we need to enable the Windows Subsystem for Linux. To do this in the search bar again type in "features" and click on the first link. Scroll down to the bottom and when you see "Windows Subsystem for Linux (Beta)" check the box and press ok. Windows will prompt you with a resart. Resart your computer and we are almost there!



Downloading And Installing Bash Terminal

Finally we are going to search again "cmd" and click the first app to appear. What we have opened is the Command Prompt, the Windows version of terminal. In the window type in bash and hit enter. Windows will now install a bash terminal on your machine, just follow the instructions given to you during the download. There are a lot of tutorials out there to help you pick up the terminal commands, however I will just be covering the ones that are necessary when needed. I do recommend the first command to run is "sudo apt update" to update your terminal. To open your terminal again search "bash" in the search bar and open the first app to appear. I also recommend to pin this to your task bar as you will be using it a lot!




Getting Started With GitHub and Git

Intro

Github is a website software developers use to share and work on code with each other. Github is the most popular tool to host open-source projects for the computer science community and is also the best source control tool as well. Github allows a code base to be shared across an entire team of developers and gives each programmer the opprotunity to be working on the project simultaneously. With each new commit to the Github repository the team will be able to see exactly what their fellow team memember changed. How is this possible? With Git of course! So on that note lets get started...

Creating a Github Account and Repository

First you are going to need to create a GitHub Account, so go on over to www.github.com and register an account. It's completely free. Now you are going to need to create a repository. This can be done by clicking on the "Create Repository" button.



Downloading Git Using Terminal

Now we are going to need to install Git to the terminal. GitHub's website has the code to copy and paste into your terminal if you are on a Mac. If you are using the bash shell on Windows 10 you are going to open up your shell and type in "sudo apt-get install git". And there you go, you have git!



Pulling Down Your Repository

Now we are going to pull down your repository so you can work on your project on your own machine. To do this first we need the url link to your repo. *See the picture to the right on where to find this* Now copy that link and head back to your bash window. We're going to navigate to the folder we want to put our project in and then use the pull command to download our project. Type in "git pull 'project link from github' ", and git will download your project onto your local machine. Now simply make edits on your project using a text editor or IDE.



Updating the Repo After You've Made Changes

It's best practice to always keep your github repository up to date! So after making changes we are going to want to go back into our terminal window and push our code back to github. So first we can use the command "git status" to check and see what is different between the repo on github and the one on our local machine. Next we use the "git add --all" commnand. This will add all of our new files and changes into the repo. Next use the command "git commit -m 'Message goes here' ". This command will commit your changes along with attaching a message to the changes. This message should be a quick summary of what you updated to the code base. Finally do "git push", which will push the commited changes back to GitHub and your repository. There all done and now you have the basic intro to Git and GitHub!

Intro To Basic Full-Stack Development And The Flask Framework

Great full-stack developers are almost always in high demand and it's extremely easy to get into. Full-stack development refers to the two main components of web based applications, which are the backend and the frontend of the website. In a quick summary the frontend is what your user interacts with. For example, say you were browsing Kayak.com for flights to Hawaii. The frontend would consist of the calender you use to create your trip window, choose between flights, and enter in your payment information. The backend on the other hand is what happens behind the scenes. Using the same example the backend is what handles your flight reservation and stores the information in a database for the airline company. The backend of a web application usually also hosts the frontend html, css, and javascript files. There are many popular backend frameworks like Java Spring framework or simply put Node.js, however in this tutorial we will be using the Python Flask framework. Flask is very simple to use and after this guide you should be able to have websites up and running in no time!
